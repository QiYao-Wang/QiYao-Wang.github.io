Question 1: 编写代码、测试代码和提交代码的基本方式

由于不提交，我这里就只进行到测试阶段。

首先应该在官方的仓库中下载 Homework 0 的相关源码，虽然不使用 mugrade 提交到平台上 但是还是建议去官方仓库中下载 mugrade, 避免其中发生不必要的依赖项报错。下载好后 在命令行里运行下面的代码，并且下载一些相关的包，其中需要用 pytest 来测试代码。 


python setup.py install
pip install pybind11 numpy numdifftools pytest

测试环境配置，需要进入 src/simple_ml.py，将其中的待填写的代码块换为 `return x + y`，之后在命令行中运行 `python -m pytest -k "add"`。 这其中的基本逻辑是 pytest 会去 tests 文件夹中寻找包含 “add” 的测试函数，在本作业中即为 tests/test_simple_ml.py 中的 test_add 函数，它 通过调用 simple_ml.py 中的 add 函数，并通过 assert 来进行正确性的验证。
